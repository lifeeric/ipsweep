# ipsweep

This is save you time and even give you best response in scanning ip which is UP!

### RUN
###### `chmod a+x ./ipsweep.sh`

### Global Access

`mv ipsweep.sh ipsweep
sudo cp ipsweep.sh /bin/`


### Example
###### `ipsweep 192.168.1`

it will give that IP address which is UP!

`192.168.1.101
19.168.1.102`
